# Python Automation Collection

> Created as a collection of Python automation experiments.

This repository contains a set of small Python scripts developed while learning automation using Python and third-party libraries.

The purpose of these projects was to explore how Python can interact with different online services and automate repetitive tasks.

---

## Projects

### 📱 WhatsApp Automation

Features:

- Send instant messages
- Schedule messages
- Send images
- Send messages to groups

**Library Used**

- PyWhatKit

---

### 📧 Email Automation

Features:

- Send emails using SMTP
- Schedule automatic email delivery

**Libraries Used**

- smtplib
- schedule

---

### 📷 Instagram Automation

Features:

- Login to an account
- Follow users
- Unfollow users
- Send direct messages
- Upload photos
- View followers and following

**Library Used**

- Instabot
